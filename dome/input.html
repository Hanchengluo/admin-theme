表单输入

注意事项

单独的表单控件会被自动赋予一些全局样式。所有设置了 .form-control 类的 <input>、<textarea> 和 <select> 元素都将被默认设置宽度属性为 width: 100%;。 将 label 元素和前面提到的控件包裹在 .form-group 中可以获得最好的排列。

预定义样式

使用下面列出的类可以快速创建一个带有预定义样式的按钮。

文件上传


<form role="form">

          <div class="form-group">

            <label for="exampleInputEmail1">邮箱输入框</label>

            <input type="email" class="form-control" id="exampleInputEmail1" placeholder="邮箱输入框">

          </div>

          <div class="form-group">

            <label for="exampleInputPassword1">密码输入框</label>

            <input type="password" class="form-control" id="exampleInputPassword1" placeholder="密码输入框">

          </div>

          <div class="form-group">

            <label for="exampleInputFile">文件上传</label>

            <input type="file" id="exampleInputFile">


          <div class="checkbox">

            <label>

              <input type="checkbox"> Check me out 

            </label>

          </div>

          <button type="submit" class="btn btn-default">Submit</button>

        </form>

水平排列的表单

通过为表单添加 .form-horizontal 类，并联合使用 Bootstrap 预置的栅格类，可以将 label 标签和控件组水平并排布局。这样做将改变 .form-group 的行为，使其表现为栅格系统中的行（row），因此就无需再额外添加 .row 了。

栅格系统与表单集合

通过为栅格系统把一行分为若干等分，在对应的等分中写入表单，在对表单添加 .form-control的class类

文本域

支持多行文本的表单控件。可根据需要改变 rows 属性。

<textarea class="form-control" rows="3"></textarea>

多选和单选框

多选框（checkbox）用于选择列表中的一个或多个选项，而单选框（radio）用于从多个选项中只选择一个。

设置了 disabled 属性的单选或多选框都能被赋予合适的样式。对于和多选或单选框联合使用的 <label> 标签，如果也希望将悬停于上方的鼠标设置为“禁止点击”的样式，请将 .disabled 类赋予 .radio、.radio-inline、.checkbox、.checkbox-inline 或 <fieldset>。

<div class="checkbox">

                  <label>

                    <input type="checkbox" value="">

                    这是一个可以操作的复选框

                  </label>

                </div>

                <div class="checkbox disabled">

                  <label>

                    <input type="checkbox" value="" disabled>

                    这是一个可以不可操作的复选框

                  </label>

                </div>


                <div class="radio">

                  <label>

                    <input type="radio" name="optionsRadios" id="optionsRadios1" value="option1" checked>

                    这是一个可以操作的单选框

                  </label>

                </div>

                <div class="radio">

                  <label>

                    <input type="radio" name="optionsRadios" id="optionsRadios2" value="option2">

                    这也是一个可以操作的单选框

                  </label>

                </div>

                <div class="radio disabled">

                  <label>

                    <input type="radio" name="optionsRadios" id="optionsRadios3" value="option3" disabled>

                    这是一个不可以操作的单选框

                  </label>

                </div>

通过将 .checkbox-inline 或 .radio-inline 类应用到一系列的多选框（checkbox）或单选框（radio）控件上，可以使这些控件排列在一行。

被禁用的输入框

为输入框设置 disabled 属性可以防止用户输入，并能对外观做一些修改，使其更直观。

<input class="form-control" id="disabledInput" type="text" placeholder="这里禁止输入..." disabled>
